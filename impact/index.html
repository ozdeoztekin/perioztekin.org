      Methodology

      How I count.

      This page is a JSON file (/data/metrics.json) rendered in your browser. Each metric carries a source — the system or document the number came from. If a value can’t be traced back to a verifiable source, it doesn’t go on this page.

      • Donations channeled are tracked through Every.org, a 501(c)(3) that lets me fundraise for partner nonprofits with full transparency. Donors give directly; funds go directly to the chosen organization.
      • People reached comes from Cloudflare Web Analytics — cookie-free, no PII, only aggregate counts.
      • Verified credentials are the cert IDs visible on the original PDFs in Academics.
      • Goals are dated commitments. The progress bars reflect concrete subtasks completed; they don’t guess.

      Anything missing from this page is missing because there’s no honest number behind it yet. As that changes, this page changes.
